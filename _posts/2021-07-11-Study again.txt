이제 막 프로모션을 만들어서, 갑자기 뭔가로 또 전환한다는 것은 좀 말이 되지 않는 것이겠지.
지금은 현재의 잡에서 하는 게 지금 시점에서 가장 잘 배워나가는 것이겠고,
<p>업데이트는 해주어야지 사람이 늘 발전이 있다고 본다. </p>
<p>마음 조급하게 가질 때는 아니다. 지금은 아니다.</p>
<p>다만 내가 꾸준히 해야 할 것은 공부이다.</p>
<p>공부는 가장 좋은 키가 되어 줄 것이다. 나의 가능성을 위한.</p>
현재 내가 해야 할 공부는 무엇인가.
일단 나는 나의 현재의 직업을 어서 qualified 시키고 싶다. 
그래서 오클라호마로 가서 배울 것을 미리 공부하는 것을 시작해야 하겠다.
그리고 지금 현재 읽고 있는 책도 꾸준히 다 읽고 싶다. Industrial Maintenance 라는 제목의 책인데, 현재 내가 하고 있는 직업에 직접적인 공부이다. 그래서 이건 좋든 싫든 꼭 해야 하는 것이고...
나는 꾸준히 업데이트를 위한 준비를 하고 싶고 그렇게 살고 싶기에,... 지금은 조급할 때는 아니지만.
공부를 해야 한다. 여러 가지의... 공부는 나에게 어떤 식으로든 도움이 되어줄 것 같다.

```
public class Main {
  public static void main(String[] args) {
    int x = 20;
    int y = 18;
    if (x > y) {
      System.out.println("x is greater than y");
    }  
  }
}
```

이건 if 에 관한 내용....
